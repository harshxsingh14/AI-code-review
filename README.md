# AI Code Review Tool (Powered by Gemini)

An automated code review system that analyzes source code, detects issues, and provides improvement suggestions using Google Gemini AI.  
This tool helps developers maintain clean, secure, and optimized code efficiently.

---

## Features

- Instant code analysis for multiple languages
- AI-generated recommendations for optimization and best practices
- Detection of:
  - syntax issues
  - performance bottlenecks
  - security vulnerabilities
  - unused imports or dead code
- Supports file upload or direct text input
- Optional storage for review history
- Usable through both UI and API

---

## Tech Stack

- Frontend: React + shadcn UI
- Backend: Next.js / Node.js
- AI Model: Google Gemini API
- Database (optional): PostgreSQL / MongoDB
- Authentication (optional): Clerk / JWT
